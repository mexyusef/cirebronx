# Code Of Conduct

This repository expects respectful, technically serious collaboration.

## Expected Behavior

- Be constructive and specific.
- Prefer evidence, logs, and reproductions over speculation.
- Treat other contributors with respect.

## Unacceptable Behavior

- Harassment or abuse
- Discriminatory behavior or language
- Repeated bad-faith disruption
- Publishing private information without consent

Maintainers may moderate or remove content that does not meet these standards.
