## Cirebronx Help

### Main Keys

`Ctrl+H`
Open or close this help dialog.

`Ctrl+B`
Toggle the right sidebar.

`Ctrl+Y`
Copy selected pane or modal content to the clipboard.

`Ctrl+X`
Quit the app.

`Tab` / `Shift+Tab`
Cycle focus forward or backward across conversation, activity, repo, and input.

### Navigation

`j` / `k`
Move selection up or down in the focused pane.

`g` / `G`
Jump to the start or end of the focused pane.

`Home` / `End`
Move cursor in input, or jump pane bounds when not editing input.

`PgUp` / `PgDn`
Move by half a pane.

`T` / `E`
Jump to the latest tool or error entry.

`Alt+B` / `Alt+F`
Move by word in the input box.

`Enter`
Inspect selected item, or submit input.

`Esc`
Close help, modal, picker, or palette.

### Input

- input is single-line and scrolls horizontally when long
- `Left` / `Right` move cursor
- `Ctrl+A` / `Ctrl+E` move to start or end
- `Ctrl+R` search prompt history backward
- `Ctrl+W` delete previous word
- `Ctrl+U` / `Ctrl+K` delete to start or end
- `Backspace` deletes backward
- `Paste` inserts raw text

### Pane Actions

`r`
Reuse selected item into the input box.

`e`
Execute selected reusable item.

`c`
Clear input.

`x`
Clear activity output.

`v` / `m` / `t`
Open provider, model, and theme pickers.

### Repo Pane

- `Enter` or `Right` expands a folder
- `Left` collapses a folder or jumps to parent
- mouse wheel scrolls
- clicking the same folder again toggles it

### Help Dialog

`PgUp` / `PgDn`
Scroll the help content.

`Home` / `End`
Jump to the start or end of help.

`Esc`
Close help.
